🏠 메인 소개

이 페이지는 기존에 만든 myClass, myProfile, myTrip, signUp 페이지를 한 곳에 모아 빠르게 이동할 수 있도록 만든 포털 사이트형 메인 Hub입니다.

오늘의 목표: 필요한 페이지를 빠르게 찾고, 나의 활동을 한눈에 확인하기

페이지 소개

  • myClass - 수업 시간표를 확인할 수 있는 페이지입니다.
  • myProfile - 나의 기본 정보와 소개를 담은 페이지입니다.
  • myTrip - 여행 사진과 미디어를 모아둔 페이지입니다.
  • signUp - 회원가입 정보를 입력하는 페이지입니다.

추천 이동 순서

  1. myProfile에서 나의 소개 확인하기
  2. myTrip에서 여행 앨범 보기
  3. myClass에서 수업 시간표 확인하기
  4. signUp에서 회원가입 페이지 확인하기
